Is Quantum Computing the End of Digital Privacy?

 

Pk encryption is one of the most popular methods of encrypting personal data and financial transactions - so its understandable to feel panicked when you hear about it being rendered obsolete by quantum computing. Quantum computers are weird and unintuitive - so it's easy for misinformation about them to spread, hopefully I can clear some things up here today.

Public keys are generated from three numbers, two of which are openly published, and one of which is known only by the intended recipient. Without getting bogged down in complicated maths - any person with the private key can use it in combination with the public numbers to decrypt the messages.

Since the public key is a combination of two public numbers and the third number, it is possible to find the private number. However, this involves finding the factors of the public key, which can take the best classical computers thousands of years.

Quantum computers however have certain properties that allow a much more efficient method of factorising to be run - this is called Shor's algorithm. The algorithm involves finding the period (or order) of a modular exponential function. The period is a value r such that a^r mod N = 1 for some integer a, where N is the modulus of the number we are trying to factor. Finding the period efficiently is the key to the algorithm's speedup.

The algorithm can only be run on quantum computers - which use qubits, which are different from normal bits because while a normal bit is either 0 or 1, a qubit can represent multiple states simultaneously. One technique uses electrons as qubits - and the changes in state represent a transition from a low energy level to a higher one. Prior to measurement the electron in in a superposition of states - it could either be in the higher or lower energy level, with different probabilities of being in each. When dealing with multiple qubits we get different probabilities of different outputs depending on the state the qubits end up assuming.

We can process these probabilities into the factors of the public key using a technique called Fourier's transform, which picks out periodic patterns in a quantum superposition. We then input these values into Shor's algorithm and voila - we have the factors of the public key, and it's a simple matter to use this information to find the private key - which would allow you to decrypt messages.

The long and short of it is that because these qubits can exist in many different states 'at once' (they exist in a superposition of states) we can investigate multiple different values at once, whereas in classical computers we would have to do it one by one. This means hundred of years worth of computation can be completed in just a few seconds, at least in theory. The largest number we've successfully factorised thus far is fifteen.

Besides - you probably care more about how this might effect your life - how for example, might we protect our private data from being decrypted. Fortunately we already have some ideas on how to circumvent this challenge:

  • Lattice based: This is based on the difficulty for computers to find the closest point to an origin - often considering many dimensions of space. Because this doesn't involve factorisation - Shor's algorithm is useless against it, though its possible an algorithm could be invented that refers this technique obsolete.
  • Isogeny based: The maths surrounding isogenies is highly complex - but they are seen as promising because of their short key length and the highly sophisticated maths behind them. They involve using the isogenies of elliptic curves to make one curve onto another.
  • Quantum encryption: quantum systems can't be read without being disturbed, so if a code is intercepted then the recipient will know that it was intercepted. This code would take the form of a stream of photons sent through a polarising filter, which could only be deciphered with a private key that can only be found by replying to the original sender - so even if the info was intercepted it would be useless to a hacker

 

Even if we do develop a secure system, companies are collecting a lot of encrypted data now in hopes of translating it in the future - so quantum computing will probably reveal at least some of our sensitive information.
